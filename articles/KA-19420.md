# Comment vérifier le stockage dans la base

## Description

Comment vérifier le stockage de la base de données pour les environnements ACC/ACS

## Résolution


Vous pouvez utiliser le Panneau de Contrôle pour vérifier le stockage de base de données utilisé ou gratuit 24/7. Reportez-vous à la section [https://experienceleague.adobe.com/docs/control-panel/using/performance-monitoring/database-monitoring/database-monitoring.html?lang=en](https://experienceleague.adobe.com/docs/control-panel/using/performance-monitoring/database-monitoring/database-monitoring.html?lang=fr) pour en savoir plus.



Remarque : vous aurez besoin d’un droit d’administrateur dans Experience Cloud pour accéder au panneau de contrôle.
