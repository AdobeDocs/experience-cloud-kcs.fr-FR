# File d’attente de réplication bloquée dans AEMaaCS


La file d’attente de réplication dans Adobe Experience Manager as a Cloud Service (AEMaaCS) peut parfois être bloquée. Suivez les étapes de cet article pour résoudre ce problème avec les autorisations d’administrateur.

## Description {#description}


### <b>Environnement</b>

Adobe Experience Manager as a Cloud Service

## Résolution {#resolution}


La suppression de la requête de l’interface utilisateur de la file d’attente de publication dorée débloquera la file d’attente. Effectuez les étapes suivantes :



1. Débloquez la file d’attente de publication dorée. Accédez au <b>AEM de démarrage</b> `>`  <b>Outils</b> `>`  <b>Déploiement</b> `>`  <b>Interface utilisateur de distribution</b> en cliquant sur la ligne du tableau Files d’attente avec le nom en gras.


2. Sur la page suivante, identifiez et enregistrez le chemin d’accès de l’élément bloquant (probablement une ressource), puis sélectionnez et choisissez de supprimer cet élément de la file d’attente. Cela permettra au reste des activations de continuer.


3. Accédez à la ressource à l’emplacement enregistré à l’étape précédente et sélectionnez <b>Gérer la publication</b>. Sélectionner <b>*Dépublier</b>* et <b>*Planification : maintenant</b>*, puis cliquez sur <b>Suivant</b>.


4. Sélectionnez uniquement la ressource sur la page <b>Portée</b> page et clic <b>Dépublier</b>. Confirmer <b>Continuer</b> si des références sont identifiées.


5. Une fois que vous avez confirmé que la ressource a été dépubliée, accédez à nouveau à la ressource et sélectionnez <b>Publication rapide</b>.


6. Vérifiez que la ressource a bien été publiée et que les files d’attente de distribution ne sont pas bloquées.
