## Description

<b>Problème :</b><br><br> <br><br>Lors d&#39;un import de fichier .zip comportant un fichier de HTML et des images dans le Concepteur d&#39;email, l&#39;erreur ci-dessous s&#39;est produite :<br><br> <br><br>*(1:117056) : Référence de caractère non valide

## Résolution


<b>Motif:</b>
