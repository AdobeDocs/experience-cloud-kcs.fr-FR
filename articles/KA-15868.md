# Erreur de publication [!DNL Analytics] segments vers Experience Cloud

## Description


Quelques raisons d’erreur affichées lors de la publication [!DNL Analytics] segments à Experience Cloud.


## Résolution


1. Description manquante : la description du segment est obligatoire pour publier une [!DNL Analytics] à l’Experience Cloud.

2. La description est trop longue : La description du segment doit comporter moins de 255 caractères.
