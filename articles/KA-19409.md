# [!UICONTROL Collecteur de fichiers] alternative d’activité dans ACS

## Description

Comme nous n’avons pas le [!UICONTROL Collecteur de fichiers] dans ACS, vous pouvez la créer à l’aide des activités suivantes et vous obtiendrez plus ou moins la même fonctionnalité.

## Résolution


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



Le [!UICONTROL Collecteur de fichiers] permet de télécharger plusieurs fichiers en fonction du modèle de nom de fichier d’un serveur externe. Chaque fichier est transmis dans la variable [!UICONTROL Chargement de fichier] , donnant des données dans la mise à jour.
Une fois l’opération terminée, il continue avec le fichier suivant, etc.
Le transfert de fichier comporte également la variable [!UICONTROL Paramètres d&#39;historisation], comme vous le savez [!UICONTROL Collecteur de fichiers].
