# Erreur : org.apache.sling.api.request.TooManyCallsException dans la console personnalisée


L’article décrit comment résoudre le problème AEMaaCS lorsque la variable *org.apache.sling.api.request.TooManyCallsException* s’affiche dans la console personnalisée. Vous pouvez soit recouvrir le chemin —` …/aem/apps/lv/console/lvjobs/content/lvjobs.html` OU définissez la limite globalement à l’aide de la propriété `'sling.max.record.requests'` sur le PID de configuration OSGi `org.apache.sling.engine.impl.SlingMainServlet`.
